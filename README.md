House Price Prediction Using Machine Learning

Project Overview

This project uses Machine Learning to predict house prices based on various features such as square footage, number of bedrooms, bathrooms, offers, brick construction, and neighborhood.

Technologies Used

- Python
- Pandas
- Scikit-learn
- Matplotlib

Machine Learning Algorithm

- Linear Regression

Features

- Data preprocessing
- Model training and testing
- House price prediction
- Performance evaluation
- Graph visualization

Files

- dataset.csv
- predictive_model.py
- graph.png
- README.md
- requirements.txt

Outcome

Successfully built a predictive model for house price estimation using Machine Learning.
